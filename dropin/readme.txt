1. Dependencies
    XAMPP Version 7.4.29
    PHP Version 7.4.29
    Supported Browser: Chrome, Mozilla Firefox, Apple Safari and Chromium
    Operating System: windows, IOS, linux

2. Installing
    XAMPP Version 7.4.29 with PHP Version 7.4.29

        Procedure to setup database
            2.1. run XAMPP application with administrator permission
            2.2. On the XAMPP application window turn on Apache and Mysql by clicking start button in actions coulmn
            2.3. Go to browser and search thr url: http://localhost/phpmyadmin/
            2.4. Click on new to create a database and set the database name as files
            2.5. click on files on left panel to select files database
            2.6. Click to import to import database attributes
            2.7. In the section File to import database choose the file files.sql, this file attached in database folder of project
            2.8. After selecting the database file click on import button 

3. Executing program
    3.1. Go (C:\xampp\htdocs\) and paste the project folder dropin
    3.2. Go to any browser and search thr url: http://localhost/dropin/
